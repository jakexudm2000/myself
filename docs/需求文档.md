# 个人技术展示网站 需求文档

## 一、项目概述
### 1.1 项目目标
打造一个高级简约风格的个人技术展示网站，用于向客户和雇主展示专业技术能力，提升个人品牌形象。

### 1.2 核心定位
- **受众**：潜在客户、雇主、合作伙伴
- **风格**：高级简约风，极简设计，突出内容
- **核心价值**：专业、可信、有技术实力的个人形象展示

---

## 二、功能需求
### 2.1 页面结构
网站采用单页滚动设计，包含以下5个核心模块：

#### 🧑‍💼 模块1：个人介绍（About Me）
- **必填内容**：
  - 姓名：Jake
  - 年龄：（待补充）
  - 性别：（待补充）
  - 职业定位：全栈工程师/技术顾问
  - 爱好：（待补充）
  - 教育经历：学历、院校、专业、毕业时间
  - 个人简介：100-200字的专业自我介绍，突出核心优势
- **交互设计**：
  - 头像hover动效：轻微放大+阴影加深
  - 文字逐字出现动画
  - 技能标签云：动态浮动效果，hover时高亮

#### 💼 模块2：工作经历（Experience）
- **内容结构**：时间线式展示，按时间倒序排列
  - 每个经历包含：公司名称、职位、任职时间、工作地点、核心职责
  - 每个经历配2-3个核心项目描述
- **动画效果**：
  - 滚动到对应区域时，时间线从左到右逐步绘制动画
  - 经历卡片从下往上滑入动画
  - 点击经历卡片时，展开详细项目描述，带折叠/展开动画
  - 项目技术标签hover时弹出详情提示

#### 🚀 模块3：项目作品集（Projects）
- **内容结构**：卡片式网格布局，每个项目包含：
  - 项目封面图/演示动图
  - 项目名称、简介、技术栈标签
  - 项目链接（Github、演示地址）
- **动画效果**：
  - 卡片hover效果：上浮+阴影加深+轻微旋转
  - 点击卡片时，弹出项目详情模态框，带淡入动画
  - 模态框内图片支持左右滑动浏览
  - 技术标签颜色区分不同技术领域（前端/后端/移动端/其他）

#### 🛠️ 模块4：技能栈（Skills）
- **内容结构**：分类展示技能，按熟练度排序
  - 前端技术：HTML/CSS/JavaScript/React/Vue等
  - 后端技术：Node.js/Python/Java/数据库等
  - 工具/其他：Git/Docker/云服务/项目管理等
- **动画效果**：
  - 技能进度条加载动画
  - 分类切换时的平滑过渡动画
  - 技能图标hover时旋转/放大效果

#### 📞 模块5：联系方式（Contact）
- **内容结构**：
  - 邮箱地址
  - GitHub链接
  - LinkedIn/掘金/知乎等技术社区链接
  - 在线留言表单（可选）
- **动画效果**：
  - 联系方式图标hover动效
  - 表单提交成功/失败的反馈动画
  - 点击邮箱自动复制到剪贴板的提示动画

### 2.2 全局功能
#### 🌓 主题切换
- 支持浅色/深色模式自由切换
- 系统默认跟随用户系统主题
- 切换主题时有平滑过渡动画
- 主题选择记忆在localStorage，下次访问自动应用

#### 📱 响应式设计
- 完美适配桌面端（1200px+）、平板（768px-1199px）、移动端（<768px）
- 移动端优化导航：汉堡菜单，折叠式内容展示
- 触摸设备优化：所有交互支持触摸操作

#### 🎨 设计规范
- **配色方案**：
  - 浅色模式：主色 #165DFF（蓝色）、背景 #FFFFFF、文字 #1D2129、辅助色 #86909C
  - 深色模式：主色 #4080FF、背景 #121212、文字 #E5E6EB、辅助色 #86909C
- **字体**：
  - 无衬线字体：Inter（优先）、系统默认无衬线字体
  - 字号层级：大标题24-32px、小标题18-20px、正文14-16px、辅助文字12-13px
- **间距**：
  - 页面左右边距：桌面端80px、平板40px、移动端20px
  - 模块间距：80-120px
  - 内容间距：16-24px
- **圆角**：卡片/按钮统一使用8px圆角，体现简约风格

#### ⚡ 性能要求
- 首屏加载时间 < 2秒
- 图片懒加载
- 代码分割，按需加载
- 静态资源CDN加速

## 三、技术选型
### 3.1 前端技术栈
| 技术 | 选型理由 |
|------|----------|
| **React 18** | 生态成熟，适合做交互丰富的单页应用 |
| **TypeScript** | 类型安全，提升代码可维护性 |
| **Vite** | 构建速度快，开发体验好 |
| **Tailwind CSS 3** | 快速实现简约风格设计，支持深色模式开箱即用 |
| **Framer Motion** | 强大的动画库，轻松实现各种交互动效 |
| **React Router** | 单页路由管理（可选，如果需要多页面的话） |
| **Lucide React** | 简约风格的图标库，符合设计风格 |

### 3.2 部署方案
- **代码托管**：GitHub 公开仓库
- **部署平台**：Vercel / Netlify / GitHub Pages（免费且自动部署）
- **域名**：建议使用自定义域名，比如 `jake.dev` 或者 `jake.tech`
- **SSL证书**：自动获取，全站HTTPS
